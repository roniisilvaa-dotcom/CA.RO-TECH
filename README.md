# CA.RO Tech

Site estatico pronto para publicacao.

## Arquivos

- `index.html`: pagina principal do site.

## Como publicar

Opcoes simples:

1. Netlify: arraste a pasta `CA.RO TECH` para o painel do Netlify.
2. Vercel: crie um projeto estatico e selecione esta pasta.
3. GitHub Pages: envie a pasta para um repositorio e publique a branch principal.
4. Hospedagem comum: envie o arquivo `index.html` para a pasta publica do servidor.
